# The AsciiDoc Editor

The AsciiDoc Editor is a stand-alone, cross-platform editor with "live"" preview of the document being edited.
It supports the full AsciiDoc markup language (<https://docs.asciidoctor.org/asciidoc/latest/>) including the more advanced
features like imports, passthrough block, conditionals, and substitutions.

## Introducution

AsciiDoc is the new kid on the block and is a very well-defined ascii bases markup language. Is is however not all users that can
work in an ascii only editor without having a preview of how the rendered output will look. This is where this asciidoc editor comes in.

## Features

- Full support for the AsciiDoc markup language
- cross-platform (Windows, Linux, MacOS)
- single executable, no installation required
- syntax highlighting in editor
- live preview of the rendered document
- Distraction free writing mode
- Built-in file browser for easy access to documents
- Support for exporting documents to HTML, PDF, and other formats using asciidoctor
- Support for custom themes and styles for the rendered output
- Support for light and dark themes for the editor
- and much more...
